Linkage Disequilibrium

Removes markers based on either pairwise or variance inflation factor LD calculations. Based on Linkage disequilibrium SNP pruning from PLINK


Batch file command:

ld


Step specific options:
-bp-max {number} Specifies a maximum chromosome specific base-pair location to process. All markers with base-pairs greater than the number will be excluded for the specified step only. Can be used in conjunction with -bp-min and -bp-space. Note: -chrom is required.

Ex: -chrom 3 -bp-max 15000000
-bp-min {number} Specifies a minimum chromosome specific base-pair location to process. All markers with base-pairs less than the number will be excluded for the specified step only. Can be used in conjunction with -bp-max and -bp-space. Note: -chrom is required.

Ex: -chrom 3 -bp-min 5000000
-bp-space {number} Specifies to process only markers spaced by {number} base-pairs across all chromosomes. Can be used in conjunction with -bp-min and -bp-max. Note: -chrom is not required.

Ex: -bp-space 1000000
-chrom {number} Specifies a specific chromosome to process. Only this one chromosome will be processed for the step specified. (Note: X = 23, Y = 24, XY = 25, Mitochondrial = 26)

Ex: -chrom 3
-ld-calc-only Perform LD r-square and d-prime calculation only (no pruning)

Ex: -ld-calc-only
-ld-chop Tells the LD step to remove the markers in LD based on either the -ld-pairwise or -ld-vif values.

Ex: -ld-pairwise 50 5 0.2 -ld-chop
-ld-pairwise {number} {number} {number} Specifies to use the pairwise LD calculation to remove markers. Argument 1: Window size; Argument 2: Window increment; Argument 3: Threshold

Ex: -ld-pairwise 50 5 0.2
-ld-vif {number} {number} {number} Specifies to use the variance inflation factor LD calculation to remove markers. Argument 1: Window size; Argument 2: Window increment; Argument 3: Threshold

Ex: -ld-vif 50 5 2
-ld-window {value} Specify number of snps to calculate at a time (default 50)

Ex: -ld-calc-only -ld-window 10
-ld-window-kb {value} Specify the basepair window span. Default is 1000kb

Ex: -ld-calc-only -ld-window-kb 500
