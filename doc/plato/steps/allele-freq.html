Allele Frequency

By default, the overall allele frequency calculation is done using founders only. If no founders in a family have genotypes at a given marker, then a random genotyped individual is chosen. You can override how the overall calculation is done by using some of the options listed below. Micro-satellite variations are included in the allele frequency output file(s), however they are included, but don't have any values in the genotypic frequency files.


Any thresholds that are specified are performed based on the overall calculation.


Batch file command:

allele-freq


Step specific options:
-all Override allele frequency overall calculation to use all samples. Cannot be used with -unaff-spouses-only, -unk-spouses, -random-child, or -all-children.

Ex: -all
-all-children Override allele frequency overall calculation to use all children. Cannot be used with -unaff-spouses-only, -unk-spouses, -random-child, or -all.

Ex: -all-children
-bp-max {number} Specifies a maximum chromosome specific base-pair location to process. All markers with base-pairs greater than the number will be excluded for the specified step only. Can be used in conjunction with -bp-min and -bp-space. Note: -chrom is required.

Ex: -chrom 3 -bp-max 15000000
-bp-min {number} Specifies a minimum chromosome specific base-pair location to process. All markers with base-pairs less than the number will be excluded for the specified step only. Can be used in conjunction with -bp-max and -bp-space. Note: -chrom is required.

Ex: -chrom 3 -bp-min 5000000
-bp-space {number} Specifies to process only markers spaced by {number} base-pairs across all chromosomes. Can be used in conjunction with -bp-min and -bp-max. Note: -chrom is not required.

Ex: -bp-space 1000000
-casecontrol Provide output borken down by male/female Case/Control.

Ex: -casecontrol
-chrom {number} Specifies a specific chromosome to process. Only this one chromosome will be processed for the step specified. (Note: X = 23, Y = 24, Mitochondrial = 25, XY = 26)

Ex: -chrom 3
-filter-file Filter markers by minor allele frequencies as found in a predefined frequency file using the -freq-file {filename} command line argument.

Ex: -filter-file
-filter-overall Filter markers by overall minor allele frequency.

Ex: -thresh-markers-min 0.1 -filter-overall
-gender Provide output borken down by overall male/female.

Ex: -gender
-group-file {filename} Specifies a sample grouping (stratification) file to use.

Columns:
FamID
IndID
Group

Ex: -group-file mygroupfile.txt
-out {name} Overrides the suffix of the output files to be {name}

Ex: -out test1
-parental Provide output broken down by male/female parents.

Ex: -parental
-random-child Override allele frequency overall calculation to use one random child from each family. Cannot be used with -unaff-spouses-only, -unk-spouses, -all-children, or -all.

Ex: -random-child
-rm-het-only Removes all snps that only have heterozygous calls

Ex: -rm-het-only
-rm-mono Filters all monomorphic markers based on the genotype data.

Ex: -rm-mono
-thresh-markers-max {number} Specifies an upper cutoff threshold for markers. All markers with values greater than the specified value will be removed from future steps.

Ex: -thresh-markers-max 0.95
-thresh-markers-min {number} Specifies a lower cutoff threshold for markers. All markers with values less than the specified value will be removed from future steps.

Ex: -thresh-markers-min 0.95
-unaff-spouses-only Override allele frequency overall calculation to use unaffected spouses only. Cannot be used with -unk-spouses, -all-children, -random-child, or -all.

Ex: -unaff-spouses-only
-unk-spouses Override allele frequency overall calculation to use unknown spouses only. Cannot be used with -unaff-spouses-only, -all-children, -random-child, or -all.

Ex: -unk-spouses
