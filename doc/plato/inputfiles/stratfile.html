Stratification File

Note: A stratification file is REQUIRED when using the output-structure step.
Specifies a stratification file to be used when creating a STRUCTURE input file.

The Structure documentation defines this field as the following: An integer designating a user-deﬁned population from which the individual was obtained (for instance these might designate the geographic sampling locations of individuals). In the default models, this information is not used by the clustering algorithm, but can be used to help organize the output.
Basically this is just a grouping of the Samples that you have into predefined population groups. If you are not sure how to define them, then just use a value of 1.

Columns required in file:
FamilyID
SampleID
Stratification value (usually numeric).


Example:
Fam123 100 1
Fam123 101 1
Fam124 100 1
Fam124 101 1
Fam223 100 2
Fam223 101 2
Fam862 1 3
Fam456 1 3
Fam231 1 3
