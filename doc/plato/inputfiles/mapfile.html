MAP Files

-map mapfile.txt


The Map file contains marker information. The order of the markers in this file should match that of the genotypes listed in the PED file.


The Map file consists of 3 columns separated by white space, defined as follows: 

     Chromosome (X is converted to 23, Y is converted to 24,
	             XY is converted to 25, 
                 mitochondrial data is assumed to be 26)
     Marker ID (typically Rsid)
     Base-pair location (base-pair relative to the chromosome)


Example: 

     1 rs1234 123456
     1 rs2345 234567
     3 rs3456 345678
     ...
