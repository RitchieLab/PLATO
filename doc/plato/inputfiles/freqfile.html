Frequency Files

-freq-file frequencies.txt

If you are wanting to load predefined minor allele frequencies, you need to specify the Marker ID (same as what is found in the MAP File) and minor alle frequency, one per line of the file. The specified markers will then have their minor allele frequencies predefined. This will currently only affect the allele-freq step. If you want to filter markers based on these values, specify -filter-file on the batch file allele-freq line.

Example file: 
     rs12354 0.1
     rs14325356 0.09
     rs234532 0.43
     ....
