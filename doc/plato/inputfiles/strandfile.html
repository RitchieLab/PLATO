Strand Flip Files

-flip strandinfo.txt

If you are wanting to flip the strands of certain markers, you need to specify the Marker ID (same as what is found in the link('MAP File','/inputfiles/mapfile');?>), one per line of the file. The specified markers will then have all of their genotypes flipped to the opposite strand.

So, if we have a marker that is specified to have the strand flipped and it's alleles are AG, then after flipping the strand, the new alleles for all genotypes of that marker would be TC. The general mapping that is performed is: 
     A => T
     C => G
     G => C
     T => A


Example file: 
     rs12354
     rs14325356
     rs234532
     ....
