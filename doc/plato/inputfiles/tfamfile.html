TFAM Files

-tfam famfile.txt


The TFAM file contains sample information when using the -tped/-tfam input method. The order of the samples in this file should match that of the genotypes listed in the TPED file.


The TFAM file consists of 6 columns separated by white space, defined as follows: 

    FamilyID
    IndividualID
    FatherID
    MotherID
    Gender
    Affection_Status


FamilyID, IndividualID, FatherID, and MotherID can all be alphanumeric.

Gender values are 1 for male, 2 for female or 0 for unknown.

Affection status values are 2 for affected, 1 for unaffected, 0 for unknown, other = and value > 2.

Example: 

     C100 S1 0 0 1 1
     C100 S2 0 0 2 1
     C100 S100 S1 S2 1 2
     ...
