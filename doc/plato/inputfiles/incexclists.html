Include/Exclude Lists

-excmarkers markerfile.txt
-incmarkers markerfile.txt
-excsamples samplefile.txt
-incsamples samplefile.txt
-excfamilies familyfile.txt
-incfamilies familyfile.txt

The include/exclude options require lists of data to include or exclude from the initial dataset upon loading of the data.

Markers

The file format for including/excluding markers is just a simple list of marker IDs that match those provided in either the Marker ID column of the MAP File or the Additional ID column of the MAP Ext File.

File format is one marker ID per line of the file: 
     Marker ID1
     Marker ID2
     Marker ID3
     ...

Example: 
     rs1234
     rs2345
     SNP_A_1234
     rs3456
     ....

Samples

The file format for including/excluding samples is just a simple list of Family ID, Sample ID pairs that match those provided in the PED File

File format is 2 columns, white space delimited: 
     Family ID
     Sample ID

Example: 
     Fam23 S3
     Fam1 S1
     Fam444 S444
     FamXYZ S1
     ....

Families

The file format for including/excluding families is just a simple list of Family IDs that match those provided in the PED File.

File format is 1 column: 
     Family ID

Example: 
     Fam23
     Fam1
     Fam444
     FamXYZ
     ....
