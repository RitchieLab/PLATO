Usage

Basic command structure: 
plato <batchfile> [option1 option2 ...]

Required arguments:
  1. A batchfile is required and should be the first argument after the 'plato' executable.
  2. A PED file and MAP file are required.
    Either specify: 
    plato batch -ped mypedfile.ped -map mymapfile.map

    or if you are working with Binary files then specify: 
    plato batch -bin-input myfileprefix

    3. For a complete list of options and more details, please see the list of command line arguments.
